## SpaceTime
Spacetime is a realtime 3D simulation that visualizes a “space fabric” bending under the mass of stars and planets, inspired by general relativity.
It is built entirely from scratch in C++, using raw OpenGL, GLEW, and GLFW, with no game engine.
